Nova Video Player
Android application privacy policy

Introduction

This privacy policy covers the use of the 'Nova Video Player' Android application.

It may not be applicable to other software produced or released by Courville Software.

Please note that Courville Software is a non-profit organization based in France, operated by volunteers.

Nova Video Player when running does not collect any statistics, personal information, or analytics from its users, other than Android operating system built in mechanisms that are present for all the mobile applications.

Nova Video Player allows videos to be played on various network transports. Cookies are not stored at any point. Authentication credentials can be stored optionally on the user's local device upon the user's explicit request. Authentication credentials might be shared across devices using proprietary encryption mechanisms provided by the Android operating system. Courville Software does not have access to any such information.

Third party cloud service dependencies

Note that Nova Video Player:
  • automatically retrieves video files metadata using The TV Database (TVDb https://thetvdb.com/) for TV shows and the The Movie Database (TMDb https://www.themoviedb.org/) cloud services for movies. The information sent to these sites does not contain any name, username, email from the user;
  • upon user request, has the ability to retrieve subtitles using opensubtitles cloud service (https://www.opensubtitles.org) with or without opensubtitles user credentials that are stored locally on the user’s device;
  • upon user activation, allows online scrobbling using trakt cloud service using the trakt user credentials that are stored locally on the user’s device.

Crash reports

When Nova Video Player application crashes, an anonymized crash report is automatically sent to Google Play developer console reporting Operating System and hardware product version details, StackTrace of the process. Additionally, beta versions of Nova Video Player include an ACRA (Application Crash Reports for Android https://github.com/ACRA/acra) module sending more detailed StackTrace with an extended history of the crash to tracepot (https://www.tracepot.com/) backend for extended debugging purposes.

Android permissions requested by the application

Note that Nova Video Player application requires the following android platform permissions:
  • “record audio” android permission in order to be able to rely on Android platform capability to perform voice search on Nova Video Player internal device media database to find a specific indexed movies or TV shows;
  • “Internal storage” access in order to be able to index local storage multimedia video files.

Nova Video Player c'est quoi?

NOVA est un Video Player open-source (opeN sOurce Video plAyer) conçu pour les produits sous Android (tablettes, téléphones et télévisions).
Cette application est une version open-source dérivée et améliorée de Archos Video Player.
NOVA est gratuit et résulte d'un effort communautaire.

Je viens juste d'installer Nova Vidéo Player et il n'affiche aucune vidéo.

Au premier lancement, la découverte de fichiers vidéos peut prendre du temps.
Nova Vidéo Player n'est pas un service de streaming, donc ne propose pas de catalogue de films.

Nova Video Player ne trouve pas d'information sur certains de mes fichiers. J'aimerais les ajouter moi-même.

Pour les utilisateurs avancés : créez un fichier NFO avec le même nom et dans le même dossier que la vidéo, puis remplissez-le en suivant les spécifications de Kodi.

Quelle est la différence entre passthrough mode 1 et passthrough mode 2 ?

Le passthrough audio permet de transmettre directement le flux audio compressé à votre système audio, qu'il décodera lui-même. Cette fonctionnalité est donc prévue pour les box TV Android.
Nova Vidéo Player offre deux choix:

- Mode 1 pour les box avec d'anciennes versions d'Android (à partir de Kitkat 4.4). Certains appareils nécessitent de régler le volume de votre box au maximum.
- Le Mode 2 est à privilégier à partir de Android 5.1 (Nvidia Shield TV et télévision Android TV).
Choisissez le mode qui vous convient le mieux selon le codec multi-channel utilisé dans vos fichiers vidéo.

Certaines vidéos saccadent sur mon appareil.

Nova Video Player dépend du décodage matériel de votre appareil. Par exemple, certains appareils ne peuvent décoder de HEVC(H265) matériellement, et utilisera le décodage logiciel qui demande plus de ressources.
